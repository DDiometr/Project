Действия, которые необходимо сделать чтобы развернуть проект:


Я использовал Apache версии 2.4, привожу действия которые нужно сделать при использовании Apache.
1)Установить Apache	.
2)Создать виртуальный хост.
3)Скачать этот репозиторий в любую папку.(дальше эту папку буду называть loadPath).
3)В папку public_html созданного виртуального хоста скопировать содержимое папки loadPath/Project/public_html.
4)В файле loadPath/Project/Script.js в 8 строке в адресе GET запроса заменить test.ru на ваш виртуальный хост.
5)Убедиться что всё работает: запустить сервер, с помощью браузера зайти на main.html созданного вами хоста, должно выглядеть так: 
https://github.com/DDiometr/worksProgramming/blob/master/Screenshot_2.png

Если всё выглядит как на скриншоте, но нет цветных квадратов в центральной области то возникли какие-то проблемы с получением файла test.txt из папки public_html, или этот файл по каким-то причинам оказался пуст(в репозитории он с данными) или его содержимое изменено.

6)В папке loadPath/Project/evoSim есть проект под QtCreator на С++, его можно использовать в этой же директории или переместить в другое место.
7)В файле main.cpp вышеупомянутого проекта нужно в 398 строке заменить путь к файлу test.txt на путь к нему на созданном вами ранее виртуальном хосте.

Порядок запуска проекта:
1)Запускаем сервер.
2)Запускаем проект на с++.
3)Заходим через браузер на файл main.html вашего хоста

Некоторые симуляции длятся очень мало, может потребоваться несколько запусков чтобы увидеть достаочно продолжительную симуляцию.
